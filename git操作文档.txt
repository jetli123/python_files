二、创建版本库
1.进入本地仓库 G:\whw_demo
2.将需要上传的文件或文件夹拷贝到whw_demo这个目录里
3.执行 git add filename or dirname   添加到索引库中
4.执行 git commit -m "描述"  提交索引库中的内容
5.执行 git push -u origin master  上传本地仓库项目到GitHub官网中的库里


退出客户端后，重新登录git.bash
首先要 同步代码：git pull
